QR Code Types

Fixed Amount QR

Pre-set payment amount for specific products or services. Perfect for menu items, retail products, or service fees.

Variable Amount QR

Customers can enter any amount when scanning. Ideal for tips, donations, or flexible payment scenarios.

Multi-use QR

Single QR code that can be used for multiple transactions. Great for table tents, business cards, or permanent displays.

Single-use QR

One-time use QR codes for specific transactions or orders. Enhanced security for high-value transactions.

Time-limited QR

QR codes that expire after a set time period. Perfect for limited-time offers or event-based payments.

Location-based QR

QR codes that only work at specific locations. Ideal for multi-location businesses or franchises.

API Integration

Generate QR codes programmatically with our simple API for dynamic business needs.

Create QR Code

curl -X POST https://api.paydaily.com/v1/qr-codes \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "type": "fixed_amount", "amount": 2500, "currency": "USD", "description": "Coffee Shop - Latte", "branding": { "logo_url": "https://yoursite.com/logo.png", "primary_color": "#2563eb", "business_name": "Coffee Corner" }, "metadata": { "location": "downtown_store", "product_id": "latte_large" } }'

Response

{ "id": "qr_1234567890", "qr_code_url": "https://api.paydaily.com/qr/qr_1234567890.png", "qr_code_data": "https://pay.paydaily.com/qr/qr_1234567890", "type": "fixed_amount", "amount": 2500, "currency": "USD", "status": "active", "created": 1640995200, "expires_at": null }
