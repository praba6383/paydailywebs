Simple Integration

Start sending payouts in minutes with our developer-friendly API and comprehensive documentation.

Single Payout Example

curl -X POST https://api.paydaily.com/v1/payouts \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "amount": 1000, "currency": "USD", "recipient": { "type": "bank_account", "bank_account": { "account_number": "1234567890", "routing_number": "021000021", "account_holder_name": "John Doe" } }, "description": "Affiliate commission payment", "metadata": { "affiliate_id": "aff_123", "commission_period": "2024-01" } }'

Bulk Payout Example

curl -X POST https://api.paydaily.com/v1/payouts/batch \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "payouts": [ { "amount": 500, "currency": "USD", "recipient": { "type": "debit_card", "debit_card": { "number": "4000000000000002" } }, "description": "Vendor payment #1" }, { "amount": 750, "currency": "USD", "recipient": { "type": "bank_account", "bank_account": { "account_number": "9876543210", "routing_number": "021000021" } }, "description": "Vendor payment #2" } ] }'
